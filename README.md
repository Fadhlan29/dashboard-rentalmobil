#  Responsive Dashboard Layout

---

To get started:

1. Clone the repository:

   ```bash
   git clone https://github.com/Risqiyyyy/rentalmobil.git

   cd Dashboard.Reactjs
   ```

2. Install the dependencies:

   ```bash
   # Using npm
   npm install
   ```

3. Start the development server:

   ```bash
   # Using npm
   npm run dev
   ```
